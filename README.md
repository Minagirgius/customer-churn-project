# 🚀 Customer Churn Prediction Dashboard

An enterprise-level Customer Churn Prediction system built using Machine Learning and Streamlit.

## 📌 Project Overview

This project was developed during the Data Mining track at ITI.

It includes:

- Data preprocessing
- Naive Bayes classification model
- ROC Curve & Confusion Matrix evaluation
- Correlation analysis
- Interactive Streamlit dashboard

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Plotly
- Joblib

## 📊 Features

- KPI Dashboard
- Interactive Visualizations
- Correlation Heatmap
- ROC Curve & Confusion Matrix
- Live Prediction Engine

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
